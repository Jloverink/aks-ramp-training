# AKS Ramp Training Records

Employee training tracking for Alaska Seaplanes ramp services.

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env` — credentials are already filled in
3. `npm install`
4. `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Add environment variables from `.env`
4. Deploy
